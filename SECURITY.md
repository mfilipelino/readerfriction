# Security Policy

ReaderFriction performs static analysis of Python source files. It does not
execute analysed code. Even so, please report any issue that could lead to
arbitrary code execution, denial of service on pathological inputs, or
information disclosure.

## Reporting a vulnerability

Open a **private** security advisory on the GitHub repository, or email the
maintainers directly if you have an address. Please include:

- A minimal reproducing input.
- The observed behaviour.
- The expected behaviour.

We aim to acknowledge reports within 7 days and to release a fix within 30
days of confirmation.
