# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/)
version 2.1. By participating in this project you agree to abide by its
terms.

## Standards

- Be welcoming and inclusive.
- Be respectful of differing viewpoints and experiences.
- Give and accept constructive feedback gracefully.
- Take responsibility for mistakes and learn from them.

## Enforcement

Report incidents to the maintainers via the project's issue tracker (mark
confidential) or by email if contact addresses are published in the README.
